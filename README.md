## 프로젝트 환경 점검

이번 수업의 모든 예제는 다음의 환경에서 진행됩니다.

* Vm Ware_ Linux Ubuntu 18.04.4 
- 버전은 크게 상관 없습니다. 하지만 수업에서 실시한 vmware& ubuntu 설치를 해주세요. 버전, bit가 다르면 많은 오류가 생길 가능성이 높습니다.
* Odroid C1 Linux Ubuntu 14.04.3
